# tunts

Linux users.

1. Download the folder Tunts

2. Inside the folder execute the command **gradle init --type basic** accepting all default options.
   
   2.1 In order to execute the aplication, you must have installed **java 1.8 or greater**.
   
   2.2 If don't have **gradle** installed, just follow the steps in the following link:
      https://gradle.org/install/

3. After that, override the file **build.gradle** generated by that in the following path:
  **src/main/resources/build.gradle**

4. Execute the command:
  **gradle run**
